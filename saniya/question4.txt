Q.How to delete one commit or multiple commits permanently Write command for it. 
Ans:
1.to remove the last commit from your Git history

	git reset --hard HEAD~1
This command will remove the last commit along with any changes in your working directory. --hard discards changes permanently.

2. Delete Multiple Commits:
to delete multiple commits,can use an interactive rebase. For example, to remove the last three commits


	git rebase --hard HEAD~3
to push commit forcefully use this command 
git push -f
